Conhecimentos

HTML

Linguagem de marcação de hipertexto. Utilizado como marcação para desenvolvimento de páginas e documentos. Garantindo a formatação ideal para sites.

JavaScript

JavaScript é uma linguagem de programação dinâmica cheia de recursos que quando aplicada em um documento HTML. Pode fornecer interatividade dinâmica em sites.

CSS

É usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. De maneira geral, com o conhecimento em CSS, formaremos uma excelente apresentação de estilos para o seu site.

C#-Sharp

Através da linguagem de programação C# é possivel, desenvolver os mais diversos sistemas e aplicativos mobile, atendendo amplos segmentos, buscando soluções rapidas e consistentes.

PostgreSQL

Um sistema de gerencimento de banco de dados, uma das pricinpais funções é armazenar dados de forma segura. Apoiando as melhores práticas, permitindo a recuperação dos dados a pedido de outras aplicações de software. Ele pode lidar com cargas de trabalho que vão desde pequenas aplicações a aplicações de grande porte voltadas para a Internet.

Git e GitHub
